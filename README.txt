An automated tool in .bat format designed for quick and bulk management of Windows Firewall rules.

It allows you to:

Scan a root folder and its subfolders.

Automatically detect all executable files (.exe).
Create inbound and outbound blocking rules in the Windows Firewall.
Automatically delete previously created rules.
Manage multiple executables simultaneously.
Run automatically with administrator privileges.

This tool is intended for:

Technical support,
Software installation,
Application isolation,
Offline testing,
Internet access control, and
System maintenance.

======================================================================================================

Herramienta automatizada en formato .bat diseñada para administrar reglas del Firewall de Windows de manera rápida y masiva.

Permite:

Escanear una carpeta raíz y sus subcarpetas.
Detectar automáticamente todos los archivos ejecutables (.exe).
Crear reglas de bloqueo de entrada y salida en el Firewall de Windows.
Eliminar automáticamente las reglas previamente creadas.
Gestionar múltiples ejecutables de forma simultánea.
Ejecutarse con permisos de administrador automáticamente.

La herramienta está orientada a:

soporte técnico,
instalación de software,
aislamiento de aplicaciones,
pruebas offline,
control de acceso a internet,
y mantenimiento de equipos.
